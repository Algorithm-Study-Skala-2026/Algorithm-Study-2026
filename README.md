# 💻 코딩테스트 대비 스터디

## 📌 스터디 규칙

| 항목 | 내용 |
|------|------|
| 🎯 목적 | 코딩테스트 대비 |
| 🗓️ 주기 | 매주 |
| ⚖️ 재판 일시 | 일요일 23:59 |
| 💸 최대 벌금 | 10,000원 |

> 목표는 본인 의지에 따라 계획 세우기.
> 단, 너무 간단하면 안됨! 팀원들이 납득할 정도로 계획 세우기 🔥




## 📋 진행 방법

1. **금주 월요일**에, 자신의 일주일 목표를 `wk00/README.md`에 공유한다
2. **금주 일요일**까지 목표치를 달성하지 못하면 **재판**을 받는다
3. 재판을 받을 때는 정해둔 목표치의 **달성도**를 보고 판단한다
   - 예: 목표 4개 중 2개만 달성 → `50% × 10,000원 = 벌금으로 내기`
   - 단, 특별한 사정이 있을 경우 팀원 토의를 통해 벌금 감면 가능
4. 목표를 **연속 달성 못 할 경우**, 벌금이 누적된다
   - 1주차: 10,000원
   - 2주차: 20,000원
   - 3주차: 30,000원 ...

### 🪧 커밋 시 참고 사항

> ⚠️ **커밋 인식 안되는 규칙** (목표 진행률에 반영되지 않음)

- 커밋 메시지에 `update` (대소문자 상관 없음) 포함 시 → 인식 안됨
- `README.md` 파일 → 인식 안됨

> 💡 이미 깃허브에 push한 파일을 수정하고 싶다면,
> 커밋 메시지에 **`update`** 를 포함시켜 주세요.
