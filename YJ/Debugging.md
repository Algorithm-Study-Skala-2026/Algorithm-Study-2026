### 1. 디버깅

> 디버깅 : 프로그램에서 발생하는 문법 오류나 논리 오류를 찾아 바로잡는 과정
> 
- 문법 오류 = 컴파일러가 자동으로 찾아줌
- 논리 오류 = 사용자의 의도와 다르게 동작하는 것 → 개발자가 찾아야 함

### 2. 문제 해결을 위한 디버깅

1. 모든 변수가 정상적으로 초기화되고 있는지 확인
2. 반복문에서 인덱스 범위 지정 오류 있는지 확인
3. 잘못된 변수 사용 오류 있는지 확인
4. 자동 형 변환 조심!
    - 몫을 구하는 `//` 연산자 이용하면 `int`가 됨
    - `/` 연산자 사용 시, `float` → 소수점의 결과까지 return
    - `%` → 나머지 값 return
